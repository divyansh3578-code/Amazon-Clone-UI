
🛒 Amazon Clone - Front-End UI
A high-fidelity, responsive recreation of the Amazon.com homepage built using pure HTML5 and CSS3. This project focuses on mastering complex layouts, navigation systems, and e-commerce UI patterns without the use of external frameworks.

✨ Key Features
Navigation Bar: A fully detailed header featuring the logo, location picker, search bar with category dropdown, and account lists.

Hero Section: A dynamic-looking hero area with background gradients and overlaying product cards.

Product Grids: Responsive layouts for product categories using CSS Grid and Flexbox.

Back to Top: A functional "Back to Top" button and a multi-column footer mirroring the original site.

Hover Effects: Interactive CSS states for buttons, links, and product images to enhance user experience.

🛠️ Tech Stack
HTML5: Semantic tags for better SEO and accessibility.

CSS3: Advanced positioning, custom properties (variables), and media queries.

FontAwesome: For high-quality scalable icons.

📱 Responsive Design
The layout is optimized for multiple screen sizes:

Desktop: Full-width navigation and 4-column product grids.

Tablet/Mobile: Adjusted search bar and stacked product cards for seamless scrolling.
